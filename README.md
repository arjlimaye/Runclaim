# RunClaim 🏃‍♂️🗺️

RunClaim is a GPS-based territory claiming app where runners physically claim hexagonal zones across their city.

Move in real life → claim tiles → expand your territory.

---

## 🚀 What it does

- 📍 Tracks your real-world movement using GPS  
- 🔷 Divides the map into hexagonal zones  
- 🏁 Lets users claim territory by physically entering zones  
- ⚔️ Compete with others for control of areas  
- 📊 Progression system (D1–D7 depth levels)

---

## 🧠 Why it’s interesting

RunClaim blends:
- Fitness + gaming
- Real-world movement + digital ownership
- Strategy + exploration

Think:
> Pokémon Go × Territory Control × Strava

---

## 🛠 Tech Stack

- **Frontend:** React Native (iOS)
- **Backend:** Supabase (Auth + Realtime + DB)
- **Core Engine:** Custom hex grid system
- **Maps & Location:** GPS tracking + map rendering

---

## ⚙️ Key Features

- Real-time syncing of territory claims  
- Custom hex grid engine (not off-the-shelf)  
- User authentication and persistence  
- Location-based gameplay logic  
- Clean modular architecture
